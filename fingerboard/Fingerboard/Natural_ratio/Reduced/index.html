Module Natural_ratio.Reduced

type natural_ratio := t
type t

A natural_ratio kept in its primed decomposition form.

val sexp_of_t : t -> Sexplib0.Sexp.t
val to_string : t -> Base.string
val equal : t -> t -> Base.bool
val one : t
val create_exn : prime:Base.int -> exponent:Base.int -> t

prime is assumed to be a prime number. The resulting t will be invalid and the behavior of the rest of the module unspecified if it isn't. The exponent is expected to be non null. Checking that prime is indeed prime is only done if its value is small enough. This should suffice in practice, since creating such values is done with small ps.

val of_small_natural_ratio_exn : numerator:Base.int -> denominator:Base.int -> t

When both the numerator and denominator and below a small bound, this function can be used to instantiate a t. Raises if out of supported bounds.
