Module Fingerboard.Roman_numeral

Manipulating small integers displayed as roman numerals. By convention, roman numerals are used to designate the vibrating strings of the instruments, with I (1) being the higher string, and increasing numbers going down. For example, the lower C string on the 4-strings cello is IV (4).

type t =
  1. | I
  2. | II
  3. | III
  4. | IV
  5. | V
include Ppx_compare_lib.Comparable.S with type t := t
val compare : t Base__Ppx_compare_lib.compare
include Ppx_compare_lib.Equal.S with type t := t
val equal : t Base__Ppx_compare_lib.equal

val to_string : t -> string
val of_int_exn : int -> t

Because we don't need to handle instruments with a very large number of strings, only a few roman numerals are representable with this module. An overflow will cause the function to raise.
